# Learn Go with tests

Using test-driven development (TDD) to learn Go.

Following: https://quii.gitbook.io/learn-go-with-tests

## Running tests

To run the tests, use the following command in your terminal:

```bash
go test ./...
```

## Project Structure

- Create modules for each section of the book
- Use the command `go mod init <module-name>` to initialize a new module
